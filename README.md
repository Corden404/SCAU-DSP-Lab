# SCAU-DSP-Lab
华农数字信号处理实验课的作业，用 Codespaces 编写与整理。
Assignments for the Digital Signal Processing lab course at SCAU. Written and organized using Codespaces.


## exp1 目录说明

`exp1` 文件夹包含两个子文件夹：

- `note_collection`：对课程 PPT 的笔记整理，便于复习与查阅。
- `skin_smoothing_demo`：基于所学库函数（如 numpy、scipy 等）实现的皮肤磨皮实验代码与演示。
